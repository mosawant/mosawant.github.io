Using the Terminal

  • Navigating the file system - GUI vs terminal
  • pwd - Print working directory
  • cd - Change directory
  • cd .. = going backwards
  • ls - List
  • cd Folder name (can use tab for autocomplete)
  • Don't need to be in an adjacent folder to cd - can use full path
  • Similar don't need to be in folder to ls - use the path and tab (remember case sensitive)
  • cd and double tab shows possible folders to navigate to
  • mkdir and rmdir to make and remove directories
  • echo "some text" > filename.txt = create file with that text
  • cp filename location = copies file to new location
  • mv filename location = moves file to new location
  • Locate feature
  • updatedb in order to locate
  • eg locate bash
  • passwd
  • man pages = instructions for different commands eg man ls will give you all the additional options such as -a for all
